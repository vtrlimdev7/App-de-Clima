# 🌦️ Previsão do Tempo Dinâmica

Um aplicativo moderno de previsão do tempo que consome dados em tempo real da OpenWeather API. Desenvolvido para praticar conceitos avançados de JavaScript Assíncrono, manipulação de DOM e persistência de dados.

## 🚀 Tecnologias Utilizadas
* **HTML5** (Estrutura semântica)
* **CSS3** (Layout responsivo e Dark Mode nativo)
* **JavaScript (ES6+)**:
  * Programação Orientada a Objetos (Classes)
  * Requisições Assíncronas (`Fetch API`, `Async/Await`)
  * Persistência de dados com `LocalStorage`

## 💡 Funcionalidades
* Busca de clima por nome da cidade.
* Exibição dinâmica de temperatura, umidade, velocidade do vento e ícones climáticos.
* Tratamento de erros (caso a cidade não seja encontrada ou o campo esteja vazio).
* Histórico de buscas recentes salvo no navegador (máximo 5 cidades).

## 🛠️ Como rodar o projeto
1. Clone o repositório.
2. Abra o arquivo `index.html` diretamente no seu navegador.
